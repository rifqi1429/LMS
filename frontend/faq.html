Diujukan (FAQ)

Temukan jawaban untuk pertanyaan paling umum tentang Skol-up LMS

Bagaimana cara login ke dashboard guru?

Kunjungi LMS.html → klik "Masuk Dashboard". Gunakan username dan password yang diberikan sekolah/admin. Jika lupa password, hubungi admin sekolah atau support kami.

Cara mengupload materi pembelajaran?

1. Login ke dashboard → pilih mata pelajaran

2. Menu "Topik & Materi" → "Tambah Materi Baru"

3. Upload PDF/video/slide → atur urutan bab → "Simpan & Publikasikan"

Format didukung: PDF, MP4, PPTX, JPG/PNG (max 100MB)

Bagaimana membuat ujian online?

Menu "Tugas & Ujian" → "Buat Ujian Baru" → Pilih tipe (Pilihan Ganda/Essay) → Tambah soal → Set timer & passing grade → Jadwalkan → Publikasikan.

Di mana melihat rekap nilai siswa?

"Rekapitulasi Nilai" → Filter kelas/mata pelajaran → Lihat grafik & detail → Export PDF/Excel untuk rapor.

Cara mengatur jadwal kelas?

Menu "Manajemen Jadwal" → Tambah pertemuan → Set tanggal/waktu → Assign materi/tugas → Notifikasi otomatis ke siswa.

Apakah ada fitur laporan hasil belajar?

Ya! Menu "Laporan Hasil Belajar" → Generate laporan per siswa/kelas → Format PDF profesional untuk orang tua & kepsek.

Jam operasional support?

24/7 via WhatsApp
Senin-Jumat: 08:00 - 20:00 WIB
Weekend: 09:00 - 17:00 WIB

Apakah support mobile app?

Ya! Responsive web app untuk Android/iOS. Login via browser → "Tambah ke Home Screen" untuk pengalaman native.
